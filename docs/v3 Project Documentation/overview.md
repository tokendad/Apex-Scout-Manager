This project folder /docs/v3 Project Documentation will contain the documents for the phases necessary for the next stage of development(V3).

Overview:  Add Scouting America to the program and all its different levels, ages and programs.   This project will become a unified program for both girls scouts and Scouting America to help manage their troops.  The Scouting America program now accepts both boy and girls.
