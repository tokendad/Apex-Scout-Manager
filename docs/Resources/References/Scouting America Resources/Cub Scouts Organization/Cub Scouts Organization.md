# Cub Scout Den Rank Chart

## Grade Level 	Rank Name
|Kindergarten|Lion|
|---|---|
|1st Grade | Tiger|
|2nd Grade	| Wolf|
|3rd Grade	| Bear|
|4th Grade	| Webelos|
|5th Grade	| Arrow of Light|


## Key Den Logistics
`Structure`: Multiple dens of various ranks combine to form a Pack.
`Tracking Progress`: Leaders often use a Den Advancement Chart to visually track which "adventures" (requirements) each Scout has completed.
`Gender Policy`: While many dens are gender-specific, "family packs" may form co-ed dens for grades K through 4. However, 5th-grade dens (Arrow of Light) must remain gender-specific to prepare for Scouts BSA.

**Usage Guidelines:**
- **Primary**: Use for rank badges, headers, and primary identification
- **Secondary/Light**: Use for backgrounds, cards, and subtle highlights
- **Tertiary/Accent**: Use for borders, accents, and emphasis elements
