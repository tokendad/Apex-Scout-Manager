## 3. UI/UX Audit (Applying Design Intelligence)

### 3.1 Visual Hierarchy
- **Issue:** The main dashboard (`view-cookies`) is text-dense and lacks a "Hero" action.
- **Suggestion:** Introduce a **Visual Goal Progress** element (SVG Ring) as the primary focal point for scouts.
- **Principle:** *120/60 Rule* â€“ One element at 120% prominence, others at 60%.

### 3.2 Cognitive Load (Booth Management)
- **Issue:** The current Booth Management workflow is contained in a 5-tab modal. This is challenging for mobile users in a "live" environment.
- **Suggestion:** Transition from a modal-heavy UX to a dedicated **"Booth Mode"** view with larger touch targets and simplified data entry.

### 3.3 "AI Aesthetic" vs. Modern SaaS
- **Issue:** Static list views and standard Bootstrap-style buttons feel functional but not distinctive.
- **Suggestion:** Apply **Glassmorphism** to cards and **Lucide Icons** consistently. Remove emoji-based icons in favor of SVG icons for a professional look.
